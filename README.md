# 🌍 AirAware Smart Air Quality Prediction System

## 📌 Overview
AirAware is a time-series based air quality forecasting system that predicts Air Quality Index (AQI) and major pollutants such as PM2.5, PM10, and NO₂ using historical environmental data.  
The project helps in identifying pollution trends and generating early warnings for unhealthy air conditions.

---

## 🚀 Features
- 📊 AQI Prediction using Machine Learning & Time Series Models  
- 🔍 Pollutant Forecasting (PM2.5, PM10, NO₂)  
- ⚙️ Data Preprocessing & Feature Engineering  
- 📈 Model Comparison (ARIMA, Prophet, LSTM, XGBoost)  
- 🚨 AQI Alert System for high pollution levels  
- 📉 Trend Analysis & Seasonal Insights  
- 🌐 Interactive Streamlit Dashboard  
- 🔄 Model Retraining with new datasets  

---

## 🛠️ Tech Stack
- **Programming:** Python  
- **Libraries:** Pandas, NumPy, Matplotlib, Scikit-learn  
- **Models:** ARIMA, Prophet, LSTM, XGBoost  
- **Visualization:** Streamlit  
- **Tools:** Jupyter Notebook, GitHub
